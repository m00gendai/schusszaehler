# Schusszähler

Ein simples UI um bei statischen SSV-Schiessprogrammen ohne elektronische Anlage die Punkte werten zu können.
Ist dafür gedacht, dass alles etwas schneller und weniger fehleranfällig abläuft und soll kein Ersatz für eine Auswertungssoftware sein.

Zur Zeit sehr rudimentär:
- Hardcoded für das Programm Schwabenkrieg-Erinnerungsschiessen Hallau Pistole 50m (2 Probe, 4 Einzel, 3 Serie 3 Serie)
- Addiert Total jeweils auf
- Schüsse können durch antippen nachträglich geändert werden

Geplant:
- Programm individuell festlegen
